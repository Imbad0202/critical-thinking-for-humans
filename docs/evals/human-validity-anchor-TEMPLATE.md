## Method

- **Author model (generated items + keys):** _e.g. claude-opus-4-8, <date>_
- **Human judge(s):** _who; competence basis in the twelve structures_
- **Item count / structure coverage:** _N items; ≥2 per structure_
- **Tier(s):** _intro / standard / advanced_
- **Blinding:** _model key hidden from the judge until after they solve_

## Epistemic status

Validity (correctness), not just stability. The number below is a
human-agreement rate; it is the primary validity statistic. Any cross-model
agreement run on the same sample is reported alongside as a secondary stability
statistic, never as a substitute. Limits: single-judge runs have no
inter-rater check; note judge count and any disagreement.
